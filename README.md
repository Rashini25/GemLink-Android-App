# GemLink – Mobile Marketplace Application

## Project Overview
**GemLink** is a mobile marketplace application designed for the gemstone industry. It connects miners, buyers, sellers, wholesalers, and jewellery manufacturers through a single digital platform.

The application aims to improve **trust, transparency, and accessibility** in gemstone trading while supporting **SDG 8 – Decent Work and Economic Growth**.

Traditional gemstone trading often relies on middlemen and physical interactions, which leads to challenges such as lack of trust, limited market access, and poor communication. GemLink addresses these issues through a centralized digital marketplace where users can interact and trade efficiently.

---

## Key Features

### Authentication
- User Registration
- Secure Login
- Profile Management

### Marketplace
- Create, Edit, and Delete Listings (CRUD operations)
- Browse gemstone products
- Category-based filtering
- Product detail view

### Communication
- In-app messaging between users
- Notifications for updates and interactions

### Trust System
- Seller ratings and reviews
- Profile-based credibility system

### Cart System
- Save items for later viewing
- Manage selected gemstones

### Subscription Model
- Free and Premium plans
- Premium features:
  - Enhanced product visibility
  - Ad-free experience

## Tech Stack
- IDE: Android Studio
- Language: Java / Kotlin
- UI Design: XML (Material Design Components)
- Backend: Firebase (Authentication, Firestore)
- Local Storage: SQLite
- Image Loading: Glide / Picasso
- Architecture: Activity-based Android architecture
